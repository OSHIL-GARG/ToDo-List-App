# 📝 Flutter To-Do List App

This is a simple yet functional **To-Do List app** built using **Flutter**.  
It allows users to:

- ✅ Add new tasks
- ❌ Delete tasks
- 🔍 Search through tasks
- ☑️ Mark tasks as done/undone
